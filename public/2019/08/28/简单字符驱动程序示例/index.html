3.1 编译

编译：当前目录下执行 make 命令，成功后生成***.ko 文件。
清除编译结果： 当前目录下执行 make clean命令。

3.2 安装及卸载

安装： sudo insmod ***.ko
卸载： sudo rmmod ***

3.3 测试

创建设备节点：
    mknod /dev/*** c 230 0
读取数据：
    cat /proc/***
写入数据：
    echo "hello world" > /dev/***
